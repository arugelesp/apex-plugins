Execute PL/SQL and Return Content Plugin
========================================

This is a Dynamic Action plugin that lets you execute PL/SQL code in the database, and then send output generated on the server (via calls to the "HTP.P" procedure) back to any element on the web page.

For details, see http://ora-00001.blogspot.com/2012/11/apex-plugin-execute-plsql-code-and-return-content-to-page.html

See also https://ora-00001.blogspot.com/2019/01/apex-plugin-execute-plsql-code-and-return-content-to-page-updated.html
